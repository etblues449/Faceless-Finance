## Daily Posting Workflow

### Step 1: Generate Script (30 min)

1. Open Faceless Finance app: https://etblues449.github.io/Faceless-Finance
2. Navigate to **Script** tab
3. Select topic (use content calendar)
4. Click **◈ Write**
5. Choose mode: Full script (recommended)
6. Review generated script:
   - ✅ Education-only tone
   - ✅ No product recommendations
   - ✅ No guaranteed returns
   - ✅ UK references (HMRC, ISA, etc.)
7. Copy script to clipboard

### Step 2: Render Avatar Video (60 min)

1. Navigate to **Avatar** tab
2. Select voice: **ElevenLabs PVC — "JB Blues Script"** (recommended)
3. Paste script into textarea
4. Review character reference (locked avatar spec)
5. Click **▶ Render Avatar Video**
6. Monitor:
   - Expect: "~60 minutes" message
   - Check **History** tab for progress
   - Status updates every 30s
7. When ready:
   - Download MP4 from History tab
   - Verify video plays
   - Verify FCA disclaimer visible (bottom overlay)
   - Verify TikTok AI label visible (🤖 badge, bottom-right)

### Step 3: Post to TikTok (15 min)

1. Open TikTok.com (upload via browser)
2. Upload MP4 (from Step 2)
3. Add caption (use template):
   ```
   [Topic hook line from your script]
   
   🇬🇧 UK personal finance education.
   
   ⚠️ This video is AI-generated using:
   🤖 Avatar: Higgsfield Soul
   🎬 Animation: Seedance 2.0
   🎤 Voice: ElevenLabs Professional Voice Clone
   
   Learn more: [link]
   
   #NotFinancialAdvice #AIGenerated #FinancialEducation #UKFinance
   ```
4. Add hashtags:
   - #UKFinance
   - #PersonalFinance
   - #TaxSaving
   - #FinancialLiteracy
5. Schedule or post now
6. Copy TikTok video URL
7. Pin FCA disclaimer as first comment (see template below)
8. Log in spreadsheet:
   - Date
   - Topic
   - Video URL
   - Post time

### Step 4: Monitor & Respond (10 min/day)

**Morning Check**:
- [ ] Videos posted successfully (no TikTok rejection)
- [ ] Disclaimer visible on preview
- [ ] No platform warnings
- [ ] New comments reviewed

**Comment Response Template**:

**If someone asks for financial advice**:
```
Thanks for asking! This content is educational only, not personal financial advice.

Please consult with an FCA-authorised financial adviser about your specific situation.

For complaints: https://www.fca.org.uk/
```

**If someone asks a clarification question**:
```
Great question! [Brief educational answer without giving advice]

For more personalized guidance, please speak with an FCA-authorised adviser.
```

**If TikTok flags for AI disclosure missing**:
```
Thanks for letting us know! The video includes:
- 🤖 AI label (bottom-right corner)
- ⚠️ AI disclosure in caption
- 📌 Pinned comment with full disclosure

All required disclosures are present.
```

## Error Handling

### If Render Fails

1. Check n8n logs for error message
2. Common causes:
   - **ElevenLabs timeout**: Retry render (usually succeeds on 2nd try)
   - **Higgsfield timeout**: Script might be too long, try splitting into 2 videos
   - **Sync quality low**: Landmark text garbled? Switch to generic London location
3. Manual fix:
   - Modify script (shorter, clearer)
   - Re-render from Avatar tab
   - Log issue for later review

### If TikTok Rejects Video

1. Common reasons:
   - **"AI generated"**: Verify AI label present + caption includes #AIGenerated
   - **"Financial advice"**: Check caption for "educational content" + disclaimer
   - **"Copyright/Music"**: No music in our videos, unlikely
2. Response:
   - Verify all disclosures present
   - Re-upload (TikTok's detection sometimes has false positives)
   - If persists: Change caption wording, try again
3. Escalation:
   - Document rejection (screenshot)
   - Email: support@tiktok.com
   - Reference: FCA compliance + C2PA metadata

### If User Comments Request Financial Advice

**Response Template**:
```
Hi! Thanks for the question. This content is educational only, not personal financial advice.

For personalized advice, please consult an FCA-authorised financial adviser.

Recommended: https://www.fca.org.uk/consumers/financial-services-guide
```
